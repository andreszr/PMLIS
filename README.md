# PMLIS
### Sistema de Préstamos MovilIs UdeA
# Pre-requisitos
- Instalar npm
- $ npm install -g sails
- $ npm install -g grunt-cli
- $ npm install -g bower

# Instalación 
- clonar repositorio
- $ npm install -> En la carpeta raiz
- $ bower install -> En la carpeta raiz

# Utilidades
- $ sails lift -> Inicia el servidor, por defecto en el puerto 1337.
